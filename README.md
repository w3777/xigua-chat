# 🍉 西瓜聊天
一个基于 Vue3 的轻量级在线聊天应用

📄 开源协议 本项目仅供学习交流使用，❗禁止任何形式的商业用途❗ 。


## ✨ 功能特性
>
> 1. 💬 发送和接收实时消息
> 2. 🟢 显示好友在线状态
> 3. 📖 已读未读消息标记
